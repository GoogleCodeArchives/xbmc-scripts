F2XTV_gui :

nom de fichier : default.py (sera lancé automatiquement lors de l'accès au dossier le contenant)

Contenu :
	- un label titre (indiquant l'écran en cours d'affichage)
	- des boutons de navigation (programmes, chaines, enregistrements ; paramètres)
	- une liste (contenant les listes des programmes, des chaines, des enregistrements et des paramètres)
	- un menu contextuel (affiché par un bouton, porte les commandes selon l'écran en cours)



Définition du menu contextuel par section :

    o Section PROGRAMMES :
        - Ajouter
        - Supprimer
        - Editer

    o Section CHAINES :
        - Ajouter
        - Supprimer
        - Regarder
        - Programmer

    o Section ENREGISTREMENTS :
        - Regarder
        - Supprimer
        - Renommer

    o Section PARAMETRES :
        - Modifier


Définition des fonctionnalités par section :
    o Section PROGRAMMES :
        - Appel d'une fenêtre (windowdialog) permettant de saisir la programmation voulue
            --> sans paramètres pour une saisie
            --> avec paramètres pour édition
                              
    o Section CHAINES :
        - ajout/suppression de chaines favorites
        - lancement de lecture d'une chaine
        
    o Section ENREGISTREMENTS :
        - renommer un enregistrement
        - déplacer un enregistrement
        - supprimer l'enregistrement
        - regarder l'enregistrement
                              
    o Section PARAMETRES :
        - configurer le chemin des enregistrements
        -


Définition du format du .ini des enregistrements :                              
[prog %chanid% %datetime%]
chanid =(int id de chaine)
datedebut = (timestamp en secondes)
duree = (int en secondes)
priorite = (int)
actionfin = (string)


Les actions de fin :
Lorsqu'un enregistrement se termine, l'utilisateur peut souhaiter :
    o Eteindre sa console : ETEINDRE
    o Etre averti par message popup : MESSAGE
    o Ne rien faire : NONE
    o Action par defaut : DEFAUT
L'action par défaut est à configurer dans les paramètres
